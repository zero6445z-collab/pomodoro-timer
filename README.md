# 🍅 番茄鐘 Pomodoro Timer

一個功能豐富的網頁版番茄鐘應用程式，幫助你提升工作效率，養成良好的工作與休息習慣。

**版本：v1.2.1** | **更新日期：2025-11-12**

## ✨ 功能特色

### 核心功能
- ⏱ **精準計時**：支援工作時段（預設 25 分鐘）、短休息（5 分鐘）、長休息（15 分鐘）
- 🔄 **自動循環**：完成 4 個番茄鐘後自動進入長休息
- 🔔 **智能提醒**：桌面通知 + 音效提醒，永不錯過休息時間
- 📊 **數據統計**：追蹤每日完成的番茄鐘數量、工作時長、休息次數
- 💾 **本地儲存**：所有數據自動保存在瀏覽器，無需註冊登入

### 🆕 v1.2.1 新功能
- 📈 **週統計圖表**：視覺化呈現工作成果
  - 過去 7 天的番茄鐘完成趨勢
  - 本週總計、平均、最佳紀錄
  - 使用原生 Canvas API 繪製
  - 自適應主題色

### 進階功能
- ⚙️ **自訂設定**：可調整各階段時長、開關音效和通知
- 🎨 **主題切換**：工作、休息模式自動切換視覺主題
- 🌙 **夜間模式**：護眼深色主題，長時間使用更舒適
- ⌨️ **快捷鍵**：空格鍵開始/暫停，R 鍵重置
- 📱 **響應式設計**：完美支援手機、平板、桌面裝置
- 🎯 **進度視覺化**：圓環進度條清楚顯示剩餘時間

## 📖 使用說明

### 基本操作
1. **開始工作**：點擊「開始」按鈕開始計時
2. **暫停**：需要暫停時點擊「暫停」按鈕
3. **重置**：點擊「重置」返回初始狀態
4. **切換模式**：點擊下方的「工作」、「短休息」、「長休息」按鈕手動切換
5. **夜間模式**：點擊右上角的 🌙/☀️ 按鈕切換日間/夜間模式

### 🆕 查看週統計
1. 在「統計」區域點擊「本週」標籤
2. 查看過去 7 天的番茄鐘完成趨勢圖表
3. 檢視本週總計、每日平均、最佳紀錄

### 自訂設定
1. 點擊右下角的 ⚙️ 設定圖示
2. 調整各階段的時長
3. 開啟或關閉音效、通知
4. 設定是否自動開始下一階段
5. 點擊「儲存設定」

### 鍵盤快捷鍵
- `空格鍵`：開始/暫停計時器
- `R`：重置計時器

## 💡 什麼是番茄工作法？

番茄工作法（Pomodoro Technique）是一種時間管理方法，由 Francesco Cirillo 在 1980 年代末期創立：

1. **選擇任務**：決定要完成的任務
2. **設定計時器**：設定 25 分鐘（一個番茄鐘）
3. **專注工作**：全神貫注工作直到計時器響起
4. **短暫休息**：休息 5 分鐘
5. **重複循環**：完成 4 個番茄鐘後，進行 15-30 分鐘的長休息

### 好處
- 提高專注力和生產力
- 減少干擾和拖延
- 維持工作與生活平衡
- 培養時間意識

## 🎯 使用技巧

1. **關閉干擾源**：在工作時段關閉社交媒體和通知
2. **明確目標**：每個番茄鐘開始前明確要完成的任務
3. **記錄中斷**：記下干擾次數，逐步改善
4. **適當調整**：根據自己的習慣調整時長
5. **充分休息**：休息時間離開螢幕，活動身體

## 🛠 自訂與擴展

### 添加自訂音效
1. 將音效文件（.mp3）放入 `assets/sounds/` 目錄
2. 在 `js/notification.js` 中修改音效載入邏輯

### 修改配色方案
在 `css/style.css` 中修改 CSS 變數：
```css
/* 日間模式配色 */
:root {
    --color-work: #E74C3C;        /* 工作模式顏色 */
    --color-short-break: #2ECC71; /* 短休息顏色 */
    --color-long-break: #3498DB;  /* 長休息顏色 */
}

/* 夜間模式配色 */
body.dark-mode {
    --color-work: #e05a4a;        /* 工作模式顏色（降低飽和度） */
    --color-short-break: #4db56a; /* 短休息顏色 */
    --color-long-break: #5aa9d6;  /* 長休息顏色 */
}
```

### 數據匯出
在瀏覽器控制台執行：
```javascript
statsManager.downloadReport()
```
會下載包含所有統計數據的 CSV 文件。

## 🐛 疑難排解

### 通知不顯示
1. 檢查瀏覽器是否允許通知權限
2. 確認設定中「桌面通知」已開啟
3. 某些瀏覽器需要使用 HTTPS 才能顯示通知

### 音效無法播放
1. 確認設定中「音效提醒」已開啟
2. 檢查瀏覽器音量設定
3. 某些瀏覽器需要使用者互動後才能播放音效

### 數據遺失
- 數據儲存在瀏覽器 localStorage 中
- 清除瀏覽器緩存會刪除所有數據
- 建議定期使用匯出功能備份數據
