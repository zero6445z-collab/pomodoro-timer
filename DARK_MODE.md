# 🌙 夜間模式說明

## 功能介紹

夜間模式是一個護眼功能，專為長時間使用番茄鐘的使用者設計。通過深色主題和降低的色彩飽和度，減少眼睛疲勞，提供更舒適的使用體驗。

## 設計理念

### 配色方案
夜間模式採用精心設計的深色配色：

**背景色調**
- 主背景：`#1a1a2e` 深藍灰色
- 卡片背景：`#16213e` 深海藍
- 文字主色：`#eaeaea` 淺灰白
- 次要文字：`#a0a0a0` 中灰色

**主題色（降低飽和度）**
- 工作模式：`#e05a4a` 柔和紅
- 短休息：`#4db56a` 柔和綠
- 長休息：`#5aa9d6` 柔和藍

### 視覺特點

1. **降低對比度**：相比日間模式的白底黑字，夜間模式使用深色背景配淺色文字，降低螢幕亮度對眼睛的刺激

2. **柔和色彩**：主題色飽和度降低約 15-20%，避免強烈色彩造成的視覺疲勞

3. **平滑過渡**：切換時有 0.3 秒的過渡動畫，避免突兀的視覺變化

4. **保持對比**：確保所有文字和元素在深色背景下仍有良好的可讀性

## 使用方式

### 手動切換
點擊頁面右上角的切換按鈕：
- 🌙 月亮圖示：點擊切換到**夜間模式**
- ☀️ 太陽圖示：點擊切換到**日間模式**

### 自動保存
您的選擇會自動保存到本地儲存，下次開啟時會自動載入您的偏好設定。

### 建議使用時機
- 🌃 **夜間工作**：晚上 8 點後使用
- 💡 **弱光環境**：在昏暗的房間使用
- 👀 **眼睛疲勞時**：長時間使用電腦後
- 🎮 **個人偏好**：喜歡深色主題的使用者

## 技術實現

### CSS 變數系統
夜間模式使用 CSS 自訂屬性（變數）實現，確保所有元素都能正確響應主題切換：

```css
body.dark-mode {
    --color-bg: #1a1a2e;
    --color-card: #16213e;
    --color-text: #eaeaea;
    --color-text-light: #a0a0a0;
}
```

### 狀態管理
- 使用 `body.dark-mode` 類別控制夜間模式
- 偏好設定儲存在 `localStorage` 中
- 頁面載入時自動讀取並應用設定

### 效能優化
- 僅切換 CSS 類別，無需重新渲染整個頁面
- 使用 CSS 過渡而非 JavaScript 動畫
- 最小化重繪和重排

## 視覺效果對比

### 日間模式
- 背景：淺灰 `#F5F5F5`
- 卡片：純白 `#FFFFFF`
- 適合：光線充足的環境

### 夜間模式
- 背景：深藍灰 `#1a1a2e`
- 卡片：深海藍 `#16213e`
- 適合：弱光或夜間環境

## 科學依據

### 為什麼夜間模式護眼？

1. **減少藍光曝露**
   深色背景發出的光線較少，特別是藍光減少約 60%

2. **降低眼睛調節負擔**
   在昏暗環境中，深色介面與環境亮度更接近，減少瞳孔頻繁調節

3. **減少眩光**
   白色背景在暗環境中會產生眩光效應，深色背景可避免此問題

4. **保持視網膜健康**
   長期曝露於高亮度螢幕可能影響視網膜健康，深色模式有助於保護

## 自訂配色

如果您想調整夜間模式的配色，可以編輯 `css/style.css` 文件：

```css
body.dark-mode {
    /* 自訂您喜歡的顏色 */
    --color-bg: #你的背景色;
    --color-card: #你的卡片色;
    --color-text: #你的文字色;

    /* 自訂主題色 */
    --color-work: #你的工作色;
    --color-short-break: #你的休息色;
    --color-long-break: #你的長休息色;
}
```

## 常見問題

**Q: 夜間模式會影響計時器功能嗎？**
A: 不會。夜間模式僅改變視覺樣式，所有功能完全正常。

**Q: 可以設定自動切換嗎？**
A: 目前版本需要手動切換。未來可能加入根據系統時間自動切換的功能。

**Q: 夜間模式下通知還會顯示嗎？**
A: 會。所有功能包括通知、音效等都正常運作。

**Q: 切換會影響數據嗎？**
A: 不會。數據儲存獨立於主題設定。

**Q: 為什麼切換後有延遲？**
A: 有 0.3 秒的過渡動畫，使切換更平滑舒適。

## 使用建議

### 最佳實踐
1. **環境配合**：在昏暗環境使用夜間模式時，建議也調低螢幕亮度
2. **定期休息**：即使使用夜間模式，也要記得每小時休息 10 分鐘
3. **燈光搭配**：建議使用柔和的環境照明，避免完全黑暗
4. **個人化調整**：根據個人感受調整螢幕亮度和夜間模式

### 護眼小貼士
- 👁 遵循 20-20-20 原則：每 20 分鐘看 20 英尺外的物體 20 秒
- 💧 多眨眼，保持眼睛濕潤
- 💡 避免在完全黑暗的環境中使用螢幕
- 🥗 攝取富含維生素 A 的食物
