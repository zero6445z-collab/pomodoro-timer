# 🍅 設定番茄圖標說明

由於 macOS 系統限制，.command 和 .bat 文件無法直接通過代碼設定自訂圖標。
需要手動設定圖標。以下是設定步驟：

## macOS 設定圖標步驟

### 方法 1：使用圖片設定（推薦）

1. **準備番茄圖標**
   - 在網路上下載番茄 PNG/ICNS 圖標
   - 或使用 Emoji：🍅（複製這個 emoji）

2. **設定圖標**
   - 在 Finder 中找到「啟動番茄鐘.command」
   - 按 `Cmd + I` 打開「簡介」視窗
   - 將圖標拖曳到左上角的小圖標上
   - 或：
     1. 選擇左上角的小圖標
     2. 按 `Cmd + V` 貼上複製的圖片

3. **使用 Emoji 作為圖標**
   - 複製 🍅 emoji
   - 按 `Cmd + I` 打開簡介
   - 選擇左上角圖標
   - 按 `Cmd + V` 貼上

### 方法 2：使用 Icon Composer（進階）

1. 下載並安裝 Xcode
2. 使用 Icon Composer 創建 .icns 文件
3. 使用上述方法套用圖標

## Windows 設定圖標步驟

### 方法：創建快捷方式

1. **創建快捷方式**
   - 右鍵點擊「啟動番茄鐘.bat」
   - 選擇「創建快捷方式」

2. **下載圖標**
   - 在網路上搜尋「tomato icon ico」
   - 下載 .ico 格式的番茄圖標
   - 儲存到 pomodoro-timer 資料夾

3. **設定圖標**
   - 右鍵點擊快捷方式
   - 選擇「內容」
   - 點擊「變更圖示」
   - 瀏覽並選擇下載的 .ico 文件

## 推薦圖標資源

### 免費圖標網站
- **Flaticon**: https://www.flaticon.com/search?word=tomato
- **Icons8**: https://icons8.com/icons/set/tomato
- **Iconfinder**: https://www.iconfinder.com/search?q=tomato
- **Freepik**: https://www.freepik.com/search?format=search&query=tomato%20icon

### Emoji 使用
最簡單的方法就是直接使用 🍅 emoji：
- macOS：可以直接貼上 emoji 到圖標
- Windows：需要轉換成 .ico 格式

## 替代方案

如果不想設定圖標，也可以：
1. 直接雙擊 index.html 使用（瀏覽器圖標）
2. 將番茄鐘加入瀏覽器書籤（使用自訂圖標）
3. 創建網頁快捷方式到桌面

---

**注意**：圖標設定是可選的，不影響番茄鐘的正常使用。
