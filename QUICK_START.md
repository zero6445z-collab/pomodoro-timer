# 🍅 番茄鐘 - 快速開始指南

## 🚀 立即使用（3 種方法）

### 方法 1：一鍵啟動（推薦）
```bash
cd ~/Desktop/pomodoro-timer
./start.sh
```

### 方法 2：直接開啟
```bash
open ~/Desktop/pomodoro-timer/index.html
```

### 方法 3：手動啟動伺服器
```bash
cd ~/Desktop/pomodoro-timer
python3 -m http.server 8000
# 然後在瀏覽器開啟 http://localhost:8000
```

## ⌨️ 快捷鍵

| 按鍵 | 功能 |
|------|------|
| `空格` | 開始/暫停 |
| `R` | 重置計時器 |

## 📖 基本使用流程

1. **點擊「開始」**開始工作
2. 工作 25 分鐘後會自動提醒
3. **休息 5 分鐘**
4. 重複 4 次後進入**長休息 15 分鐘**

## ⚙️ 自訂設定

1. 點擊右下角 ⚙️ 圖示
2. 調整時長、音效、通知等設定
3. 點擊「儲存設定」

## 📊 查看統計

主畫面下方實時顯示：
- 今日完成番茄鐘數
- 總工作分鐘數
- 休息次數

## 🎯 使用技巧

- 🔕 工作時關閉所有干擾
- 📝 每個番茄鐘前明確目標
- ☕ 休息時離開螢幕
- 📈 定期查看統計數據
- 🌙 夜間使用建議開啟夜間模式（點擊右上角 🌙/☀️）

## 🐛 常見問題

**Q: 通知沒有顯示？**
A: 檢查瀏覽器是否允許通知權限

**Q: 音效無法播放？**
A: 確認設定中「音效提醒」已開啟

**Q: 數據會保存嗎？**
A: 會！數據自動保存在瀏覽器本地
