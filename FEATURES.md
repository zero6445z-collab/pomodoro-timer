## 使用場景

### 場景 1：標準番茄工作法
```
1. 點擊「開始」→ 工作 25 分鐘
2. 收到通知 → 休息 5 分鐘
3. 重複 3 次
4. 第 4 次後 → 長休息 15 分鐘
```

### 場景 2：自訂時長
```
1. 打開設定
2. 調整為：工作 50 分鐘 / 休息 10 分鐘
3. 儲存設定
4. 適合需要長時間專注的任務
```

### 場景 3：夜間使用
```
1. 點擊右上角 🌙 開啟夜間模式
2. 享受舒適的深色介面
3. 設定自動保存
4. 下次開啟自動載入
```

### 場景 4：查看統計
```
1. 完成多個番茄鐘
2. 查看主畫面下方統計
3. 了解今日生產力
4. 可匯出詳細數據
```
