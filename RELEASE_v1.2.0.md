## 🆕 新功能亮點

### 1. 📋 待辦清單整合

再也不用在多個應用之間切換！現在您可以直接在番茄鐘中管理任務：

✨ **核心功能**
- 輕鬆新增、完成、刪除任務
- 智能篩選：查看全部/進行中/已完成任務
- 自動追蹤每個任務完成的番茄鐘數
- 一鍵清除所有已完成任務

🎯 **完美整合**
- 每次完成番茄鐘，當前任務的計數自動增加
- 直觀了解每個任務投入的時間
- 所有數據自動保存，永不丟失

### 2. 🎵 白噪音播放器

專注力提升利器！6 種精心設計的環境音助您進入心流狀態：

🌧️ **雨聲** - 柔和的雨滴聲，營造平靜氛圍
🌊 **海浪** - 舒緩的波浪聲，放鬆心情
🌲 **森林** - 鳥鳴與樹葉沙沙，親近自然
🔥 **壁爐** - 溫暖的木柴燃燒聲，冬日首選
☕ **咖啡廳** - 輕鬆的環境氛圍，如同身處咖啡店
💨 **風聲** - 自然的風吹聲，清新怡人

⚡ **技術特色**
- 使用 Web Audio API 程序化生成，無需下載音頻文件
- 獨立音量控制，精確調節
- 設定自動保存，下次打開保持您的偏好

### 3. 📈 週統計圖表

數據視覺化讓成果一目了然！

📊 **精美圖表**
- 過去 7 天番茄鐘完成趨勢長條圖
- 清晰顯示每天的工作量
- 使用原生 Canvas API 繪製，流暢無延遲

📈 **智能統計**
- 本週總計：查看本週總共完成多少個番茄鐘
- 每日平均：了解您的平均工作效率
- 最佳紀錄：激勵您創造更好的成績

🎨 **完美適配**
- 自動使用當前主題色
- 夜間模式下使用舒適的深色配色
- 標籤式切換，輕鬆查看今日/本週統計

## 🚀 快速開始

### 線上使用
直接開啟 `index.html` 即可使用，無需安裝！

### 推薦使用流程

1. **設定任務**
   - 在待辦清單中新增今天要完成的任務
   - 按優先級排序

2. **選擇環境音**
   - 根據心情選擇白噪音
   - 調整到舒適的音量

3. **開始工作**
   - 點擊「開始」按鈕
   - 專注 25 分鐘
   - 完成後任務計數自動增加

4. **查看成果**
   - 切換到「本週」標籤
   - 查看您的工作趨勢
   - 為明天設定更好的目標
